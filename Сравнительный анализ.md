# Сравнительный анализ систем Яндекс.Геокодер, DaData, GigaChat 
## Цель: Сравнить системы Яндекс.Геокодер, DaData, GigaChat для принятия решения об использовании сервиса прямого/обратного геокодирования
** В рамках задачи было проведено тестирование прямого и обратного геокодирования. Проверено 200 точек на карте Российской Федерации с помощью систем Яндекс.Геокодер, DaData, GigaChat. В  работе использованы API всех сервисов, выполнялись запросы к API систем через Postman, а также с помощью адресной строки браузера. Проведен анализ статистики совпадений по прямому и обратному геокодированию **

**Результаты тестирования в виде файла Excel приложены к данной работе в репозитории GitHab**

## В результате из 200 адресов верно определено следующее количество:
- **Яндекс. Геокодер** : 200 адресов
- **DaData**: 17 адресов
- **GigaChat**: 3 адреса

## Комментарии:
- **Яндекс. Геокодер** : Корректно и оперативно определяет адреса как при прямом, так и при обратном геокодировании
- **DaData**: Допускает неточности как при прямом, так и при обратном геокодировании
- **GigaChat**: Допускает критические ошибки как при прямом, так и при обратном геокодировании

## Вывод: Только Яндекс. Геокодер рекомендуется использовать в качестве сервиса прямого/обратного геокодирования
